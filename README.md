# wurstball-viewer


#### 05.04.2016
 - added presentation mode

Controls:

**Key** | **Description**
-----|------
r | random image
+ | zoom in
- | zoom out
CTRL + left | previous image
CTRL + right | next image
CTRL + C | copy image address to clipboard
CTRL + S | save image
CTRL + F | toggle full screen mode
SPACE | start/stop presentation mode
